

## Project Overview
This project focuses on analyzing sales performance to identify key trends, top-performing products, and regional growth opportunities. Using Python and data visualization, I transformed raw sales data into actionable business insights.

## Tools & Technologies
* **Language:** Python 3
* **Libraries:** Pandas (Data Cleaning), Matplotlib/Seaborn (Visualization), NumPy
* **Environment:** Jupyter Notebook
* **Dataset:** `cleaned_sales_data.csv`

## Key Analysis Features
In the `Data_Analytics_Project.ipynb` notebook, I performed the following:
1.  **Data Cleaning:** Handled missing values and formatted date columns for time-series analysis.
2.  **Exploratory Data Analysis (EDA):** Identified monthly sales trends and peak seasons.
3.  **Product Performance:** Ranked products by total revenue and quantity sold.
4.  **Visualizations:** Created bar charts and line graphs to display growth patterns.
